# PAC1-HTML-CSS-1
=====================

## Creación del boilerplate

Creación del archivo package.json
Configuración de valores por defecto de la instrucción npm init --yes
Instalación de Parcel y Rimraf
Configuración del archivo package.json para el uso de Parcel
    - Comportamiento del hot loader y live server
    - Publicación de archivos
    - Uso de javascript ES6
    - Uso de pre y post procesadores: babel, postHTML, postCSS y Autoprefixer
    - Compatibilidad entre navegadores
